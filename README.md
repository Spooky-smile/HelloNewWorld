# 💬 关于
---
### 使用说明
#### 确定
1. 在 “计划一” 于 “计划二” 中键入你想做但不知道先做哪个的个计划
2. 单击 “确定” 软件将在新窗口中输出结果 
#### 重置
1. 输入完两个计划后，如需同时清除，可单击 “重置” 以重置
2. 也可用于多计划的抉择
#### 退出
1. 欲关闭软件运行窗口可单击 “退出” ，其效果与单击右上角“ ❌ ”相同
### 环境
|编程平台|使用语言|软件可用运行格式|
|-|-|-|
|Windows|Python3.7.2|exe、py|
